# FAQ Sistem Informasi Operasional Sumur
Dokumen ini berisi pertanyaan dan jawaban yang sering muncul saat menggunakan Sistem Informasi Operasional Sumur

## Saya tidak bisa login ke dalam sistem, apa yang harus saya lakukan?
Pastikan email dan password sudah terisi dengan benar. Jika masih tidak bisa login silakan hubungi admin
untuk melakukan pengecekan akun dan hak akses.

## Data yang saya input tidak bisa disimpan, apa yang harus saya lakukan?
Pastikan semua data wajib sudah terisi dengan benar. Jika data masih tidak tersimpan,
cek kembali koneksi jaringan internet atau hubungi admin.

## Sistem tidak bisa melakukan pengunduhan data Excel, apa penyebabnya?
Pastikan jenis sumur serta tanggal awal dan akhir sudah terpilih dengan benar. Jika proses export masih gagal, cek kembali koneksi internet atau pastikan server lokal (XAMPP) berjalan dengan baik.
