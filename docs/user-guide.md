# User Guide Sistem Informasi Operasional Sumur

## Tujuan Dokumen
Dokumen ini bertujuan untuk memberikan panduan cara menggunakan Sistem Informasi Operasional Sumur agar lebih mudah 
dipahami.

## Prasyarat
1. Pengguna harus memiliki email dan password untuk login
2. Pengguna memiliki hak akses yang sudah diberikan oleh sistem  
3. Sistem telah dijalankan menggunakan XAMPP sebagai server lokal
4. Browser web telah terpasang ( Mozilla Firefox atau Google Chrome)
   
## Alur Penggunaan Umum
1. Pengguna melakukan login ke sistem
2. Pengguna meinput data operasional sumur
3. Sistem menyimpan data ke dalam database
4. Pengguna dapat melihat kembali data yang sudah disimpan
   
## Langkah Penggunaan
### 1. Login
1. Pengguna masuk ke halaman login 
2. Pengguna memasukkan email dan password
3. Klik tombol **Login**
4. Sistem menampilkan halaman utama
   
### 2. Input Data Operasional Sumur
1. Pilih menu **Tambah Data**
2. Isi tanggal, debit, kadar, jam kerja, jumlah air
3. Klik tombol **Simpan**
4. Sistem menyimpan data operasional sumur
   
### 3. Melihat Rekap Data
1. Pilih menu **Rekap**
2. Pilih tanggal awal, tanggal akhir dan jenis sumur
3. Klik tombol **Export**
4. Sistem menampilkan hasil berupa file Excel
   
## Catatan Penting
1. Pastikan semua data sudah terisi dengan benar sebelum disimpan
2. Hubungi admin jika ada gangguan hak akses
